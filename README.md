- [Livewire](https://livewire.laravel.com/)

## Installation

- ```composer create-project --prefer-dist edyspider/laravel-10-starter-kit```.
- Run ```php artisan key:generate```.
- Edit database connection in ```.env``` if needed.
- Run ```php artisan migrate```
- Run ```npm install```

## Usage

### Development

```npm run dev```

### Production

```npm run build```
